# Connect Zip

A lightweight Connect Four-style game implemented with React and Vite.

## Play
- Click a column to drop a disc.
- Keyboard: Left/Right to move, Space/Enter to drop, R to reset.

## Dev

```sh
pnpm i
pnpm -w -C apps/zip dev
```

## Build

```sh
pnpm -w -C apps/zip build
pnpm -w -C apps/zip preview
```
